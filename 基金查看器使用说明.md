# 基金查看器使用说明

这是一个在终端查看基金信息的Python脚本，支持实时监控、基金搜索、详细信息查看等功能。

## 安装依赖

```bash
pip install -r requirements.txt
```

或者单独安装：
```bash
pip install requests colorama
```

## 基本用法

### 1. 查看基金基本信息
```bash
python fund_viewer.py 000001
python fund_viewer.py 000001 000300 161725  # 同时查看多个基金
```

### 2. 查看基金详细信息
```bash
python fund_viewer.py 000001 -d
python fund_viewer.py 000001 000300 -d
```

### 3. 搜索基金
```bash
python fund_viewer.py -s 华夏
python fund_viewer.py -s 沪深300
python fund_viewer.py -s 科技
```

### 4. 实时监控基金
```bash
python fund_viewer.py 000001 -w              # 默认30秒刷新
python fund_viewer.py 000001 -w -i 10        # 10秒刷新
python fund_viewer.py 000001 000300 -w -i 5  # 监控多个基金，5秒刷新
```

## 功能特性

- ✅ **实时数据**: 获取基金实时净值和估算净值
- ✅ **彩色显示**: 涨跌用红绿色区分，界面友好
- ✅ **基金搜索**: 支持基金名称和代码搜索
- ✅ **详细信息**: 基金经理、公司、类型、规模等
- ✅ **实时监控**: 支持多基金实时监控
- ✅ **多基金查看**: 一次查看多个基金
- ✅ **错误处理**: 完善的错误提示和异常处理

## 显示信息说明

### 基本信息
- **基金代码**: 6位数字基金代码
- **基金名称**: 基金完整名称
- **最新净值**: 最新公布的单位净值
- **估算净值**: 实时估算的净值
- **涨跌幅度**: 当日涨跌幅度百分比
- **更新时间**: 数据最后更新时间

### 详细信息（使用 -d 参数）
- **基金经理**: 当前基金经理姓名
- **基金公司**: 基金管理公司
- **基金类型**: 基金分类类型
- **成立日期**: 基金成立时间
- **基金规模**: 基金总资产规模

## 使用技巧

1. **快速查看自选基金**：
   ```bash
   python fund_viewer.py 000001 000300 161725
   ```

2. **实时监控重点基金**：
   ```bash
   python fund_viewer.py 000001 000300 -w -i 10
   ```

3. **搜索感兴趣的基金**：
   ```bash
   python fund_viewer.py -s 科技
   python fund_viewer.py -s 医疗
   ```

4. **查看基金详细信息**：
   ```bash
   python fund_viewer.py 000001 -d
   ```

## 注意事项

- 数据来源于东方财富网，请以官方数据为准
- 估算净值仅供参考，实际净值以基金公司公布为准
- 基金投资有风险，请谨慎决策
- 监控模式下按 `Ctrl+C` 退出

## 故障排除

1. **网络连接问题**：检查网络连接是否正常
2. **基金代码错误**：确认基金代码是否正确（6位数字）
3. **依赖包问题**：确保已安装 requests 和 colorama
4. **权限问题**：确保有执行 Python 脚本的权限
